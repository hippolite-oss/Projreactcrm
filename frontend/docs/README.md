# Documentation du Projet CRM

Ce dossier contient tous les fichiers de documentation, tests et configuration du projet CRM.

## Structure

- `project-files/` - Fichiers de configuration et documentation du projet
- `test-files/` - Fichiers de test et prototypes
- `config-files/` - Fichiers de configuration IDE et outils

## Fichiers Déplacés

Tous les fichiers qui étaient à la racine du projet ont été organisés ici pour garder une structure propre avec seulement les dossiers `backend/` et `frontend/` visibles à la racine.
