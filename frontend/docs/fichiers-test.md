# Fichiers de Test du Projet

Ce document répertorie les fichiers de test qui étaient à la racine du projet et qui ont été organisés pour une meilleure structure.

## Fichiers HTML de Test
- `test-clients-tailwind.html` - Test de l'interface clients avec Tailwind CSS
- `test-jspdf.html` - Test de génération PDF avec jsPDF
- `test-notifications-z-index.html` - Test du z-index des notifications
- `test-reports-fixes.html` - Test des corrections de rapports

## Fichiers JavaScript de Test
- `test-email-api.js` - Test de l'API email
- `test-email-simple.js` - Test d'envoi d'email simple
- `test-i18n-backend.js` - Test de l'internationalisation backend
- `test-reports-api.js` - Test de l'API des rapports

## Notes
Ces fichiers de test peuvent être utilisés pour valider différentes fonctionnalités du système. Ils sont maintenant organisés dans ce dossier docs pour une meilleure structure du projet.
